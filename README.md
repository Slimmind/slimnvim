# SLIMNVIM

## Installation

1. install nvim

2. go to the ~/.config folder

   `cd ~/.config`

3. create nvim folder

   `mkdir nvim && cd nvim`

4. initialize git repository

   `git init`

5. add remote repository with configuration

   `git remote add origin git@github.com:Slimmind/slimnvim.git`

6. create master branch and pull

   `git checkout -b master && git pull origin master`

7. install necessary npm-packages

   `npm install -g pyright`

8. start nvim

   `nvim`
